### 레벨 3 중간 회고

### 수면 시간 지키기

어느 순간 잠을 제대로 못 자는 것 같다.  
처리해야 할 일은 없었지만 고민이 많아서 그런지 깊게 자는 시간이 많이 줄어든 것 같다.  
남은 레벨 3 동안은 하루에 6시간은 꼭 잠을 자도록 해야겠다.  

### 칭찬하는 습관 들이기

타인의 장점을 발견해 칭찬하는 것, 중요하지만 쉽지 않을 때가 많다.  
작은 것이라도 진심을 다한 칭찬을 하는 건 필요하다.  
존중하고, 칭찬하는 것에 익숙해지자.  

### 학습의 비효율 제거하기

어느 순간 어떤 걸 학습해야 할지 갈피를 못 잡고 있는 것 같다.  
목표가 없으니 늘어지는 느낌이 많이 든다.  
JPA를 조금 더 깊게 공부하는 시간을 가져야겠다.  

### 발표 준비에 시간을 들이기

남들 앞에 서는 게 익숙하지 않은 것 같다.  
익숙하지 않은 만큼 조금 더 시간을 써 준비를 해야겠다.  
이번에 데모데이 발표를 준비하는 시간이 너무 짧았지만, 더 잘할 수 있지 않았을까?
